-   주제 : 당근마켓 클론코딩

### Usage

---

1. Git Fork

2. Git clone

```
git clone https://github.com/eunnnii/web-front.git
```

3. npm install

```
npm install
```

4. yarn install

```
npm install yarn
```

5. start

```
yarn start
```

or

```
npm start
```
